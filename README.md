# Бесплатный прокси через render

- Регистрируемся на [рендере](https://render.com) удобным для вас способом (почта, гугл, гитхаб, гитлаб). 
- Создаем новое рабочее пространство, пропускаем обучение
- В дашборде нажимаем на +, создаем Web Service
- Выбираем Публичный гит-репозиторий, вставляем в поле ссылку https://github.com/w9line/render_socks5proxy , нажимаем connect.
- Даем уникальное имя, по которому в дальнейшем будет производится подключение
- Выбираем регион и бесплатный тариф
- Листаем вниз, деплоим

## Подключение к прокси 

Для подключения необходимо скачать [gost](https://github.com/go-gost/gost/releases) 

после установки : 
``gost -L socks5://:PORT -F "socks5+wss://user:pass@WEBAPPLINK(без http):443" ``
после  этой команды на указанном порту будет поднят сокс5 прокси, который можно использовать где угодно.
